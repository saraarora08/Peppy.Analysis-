# Peppy.Analysis-
Performed data analysis on user onboarding and engagement patterns based on real internship experience at Peppy, a wellness platform focused on intimate and personal care. The analysis aimed to identify drop off points in the user journey and understand factors affecting user engagement and product adoption in a sensitive product category. Leveraged SQL to query and analyze structured data, enabling efficient extraction of insights. The findings highlight opportunities to improve onboarding flow, reduce user hesitation, and enhance platform adoption through better user experience and engagement strategies.
